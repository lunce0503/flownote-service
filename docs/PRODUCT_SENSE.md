# Product Sense

## Product Intent

Flownote는 작업, 노트, 캔버스, 에이전트, 소셜, 주식 관리를 한 로컬 통합 환경에서 다루는 생산성 앱이다. 핵심 방향은 사용자가 학습과 작업 흐름을 기록하고, 에이전트가 그 맥락을 읽어 다음 행동을 돕는 것이다.

## Primary User Flows

- 계정 생성과 로그인 후 개인 데이터 사용
- 작업과 반복 시간표 관리
- 노트 작성과 폴더/카테고리 정리
- 캔버스 기반 드로잉 필기와 이미지 업로드
- 에이전트 대화를 통한 노트/작업 맥락 정리
- 소셜 톡방과 메시지 관리
- Yahoo Finance 기반 주식 자산과 차트 확인
- Expo 모바일 앱에서 WebView로 같은 서비스 접근

## Product Principles

- 기능은 첫 화면에서 바로 사용 가능해야 한다.
- 데이터가 비어 있을 때도 다음 행동을 명확히 보여준다.
- 에이전트 기능은 사용자 데이터의 실제 맥락을 읽고 행동 후보를 제안해야 한다.
- 모바일은 데스크톱 기능의 축소판이 아니라 작은 화면에서 완료 가능한 흐름이어야 한다.
- 외부 API 기능은 실패해도 앱의 기본 데이터 관리 기능을 깨지 않아야 한다.

## Success Criteria

- 사용자는 작업, 노트, 시간표, 캔버스, 주식 정보를 새로고침 후에도 유지할 수 있다.
- 주요 화면은 인증 실패, 데이터 없음, 서버 오류를 명확히 표시한다.
- 에이전트가 읽을 수 있는 데이터 형태가 안정적이고 문서화되어 있다.
- 모바일에서 서버 URL, QR, WebView 연결 흐름이 문서와 일치한다.

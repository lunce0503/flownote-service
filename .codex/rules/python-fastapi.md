# Python FastAPI 규칙

- `flownote-API/`에서는 `uv`와 프로젝트 lockfile을 사용한다.
- 서비스 모듈이 이미 있으면 route handler는 얇게 유지한다.
- 요청과 응답 형태는 타입이 있는 모델로 검증한다.
- 주변 코드가 async라면 DB 접근도 async를 우선한다.
- 환경 변수 이름은 명시적으로 유지하고, 인증 정보나 호스트를 하드코딩하지 않는다.
- 특정 엔드포인트에서만 필요한 무거운 선택 의존성은 모듈 import 시점에 불러오지 않는다.
- 외부 API 실패는 관찰 가능하게 만들되 비밀값은 노출하지 않는다.
- MCP 서버에서는 도구 입력을 신뢰하지 말고, 파일시스템/네트워크/DB 접근 전에 검증한다.

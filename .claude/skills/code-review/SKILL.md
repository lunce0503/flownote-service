name: code-review
description: 코드 리뷰는 개발자가 작성한 코드를 다른 개발자가 검토하여, 코드의 품질, 정확성, 보안, API 계약 준수, 테스트 커버리지 등을 평가하는 과정입니다. 이 리뷰는 코드 변경 사항이 프로젝트의 기준과 요구사항을 충족하는지 확인하고, 잠재적인 문제를 조기에 발견하여 수정할 수 있도록 돕습니다.
---

1. diff와 수정된 파일을 확인한다.
2. 정확성, 회귀, 보안, API 계약, 누락된 테스트를 우선한다.
3. findings를 심각도 순서로 먼저 제시한다.
4. 가능하면 파일과 줄 참조를 포함한다.
5. 요약은 짧고 부차적으로 둔다.

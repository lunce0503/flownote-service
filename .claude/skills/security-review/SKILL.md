---
name: security-review
description: 보안 리뷰는 코드 변경 사항이 시스템의 보안에 미치는 영향을 평가하는 과정입니다. 이 리뷰는 코드에서 잠재적인 보안 취약점이나 위험 요소를 식별하여, 이를 수정하거나 완화할 수 있도록 돕습니다. 보안 민감 변경에 이 스킬을 사용한다.
--- 

확인 항목:

- 비밀값 노출.
- 인증 및 권한 우회.
- SQL injection.
- Path traversal.
- 안전하지 않은 업로드.
- SSRF 또는 안전하지 않은 outbound request.
- 안전하지 않은 MCP/tool input.
- 과도하게 넓은 환경 가정.

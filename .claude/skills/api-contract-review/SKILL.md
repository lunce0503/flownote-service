---
name: api-contract-review

description: API 계약 리뷰는 API의 생산자와 소비자 간의 계약을 검토하여, API가 예상대로 작동하는지 확인하는 과정입니다. 이 리뷰는 API의 요청과 응답이 명세에 부합하는지, 그리고 모든 관련 프로젝트에서 API 호출이 올바르게 이루어지고 있는지를 검증하는 것을 포함합니다.
---
1. 생산자와 소비자를 식별한다.
2. 요청 검증과 응답 형태를 확인한다.
3. 모든 하위 프로젝트에서 호출부를 검색한다.
4. DB 가정을 검증한다.
5. 계약에 의존하는 테스트, 타입, 문서를 업데이트한다.
